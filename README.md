# Encriptador Desencriptador

Este proyecto es una aplicación web que permite encriptar y desencriptar textos. La aplicación fue diseñada utilizando HTML, CSS y JavaScript.

## Características

- *Encriptar texto*: Convierte las vocales de un texto a códigos específicos.
- *Desencriptar texto*: Reconvierte los códigos a las vocales originales.
- *Copiar texto*: Permite copiar el texto encriptado/desencriptado al portapapeles.
- *Interfaz responsiva*: Diseñada para adaptarse a diferentes tamaños de pantalla (escritorio, tablet, móvil).

## Instrucciones de uso

1. *Encriptar*: 
   - Ingresa el texto que deseas encriptar en el cuadro de texto de la izquierda.
   - Haz clic en el botón "Encriptar".
   - El texto encriptado aparecerá en el cuadro de la derecha, y podrás copiarlo haciendo clic en el botón "Copiar".

2. *Desencriptar*:
   - Ingresa el texto encriptado en el cuadro de texto de la izquierda.
   - Haz clic en el botón "Desencriptar".
   - El texto original aparecerá en el cuadro de la derecha.

3. *Copiar*:
   - Después de encriptar o desencriptar, haz clic en el botón "Copiar" para copiar el texto al portapapeles.

## Tecnologías utilizadas

- *HTML5*: Para estructurar la página.
- *CSS3*: Para el diseño y la disposición de los elementos.
- *JavaScript*: Para la lógica de encriptado/desencriptado y la interacción con el usuario.

## Estructura del proyecto

- index.html: Contiene la estructura principal de la aplicación.
- style.css: Archivo de estilos CSS.
- app.js: Contiene la lógica de encriptado, desencriptado y copiado.
- imagenes/: Carpeta que contiene las imágenes utilizadas en la aplicación.
